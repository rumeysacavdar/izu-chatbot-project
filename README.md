\# IZU Chatbot Project



This project is an AI-powered chatbot developed for Istanbul Sabahattin Zaim University.



\## Features



\- Answers questions about university professors

\- Provides information from selected IZU web pages

\- Uses collected university links as knowledge source

\- Supports student-related questions

\- Can be expanded with new links and documents



\## Technologies Used



\- Python

\- Flask / Streamlit

\- OpenAI API

\- LangChain

\- ChromaDB / Vector Database

\- HTML / CSS
